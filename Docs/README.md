# KSM Racing – Personal Website

## Project name
**KSM Racing** – the personal racing profile website for Kseniia Miliaeva.

## What this website is
A simple personal website that shows Kseniia Miliaeva as a 15-year-old karting
driver. It shares her story, racing experience, the tracks she races at, a photo
gallery and information for possible sponsors. It is built with plain **HTML** and
**CSS** as a beginner learning project.

## Live website
The site is published on **Azure Static Web Apps**:
https://brave-bay-0f2aa9103.7.azurestaticapps.net

It deploys automatically: every push to the `main` branch on GitHub updates the
live site through a GitHub Actions workflow.

## How to open the website (Live Server)
1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension (if you have not already).
3. Open the website file you want to view (see below).
4. Right-click inside the file and choose **"Open with Live Server"**.
5. The website opens in your browser. When you save a change, it refreshes
   automatically.

## Which file is the main version
The main (first) design is here:

- `Codes/personal-site.html` – the page
- `Codes/personal-style.css` – the styles

This is the version to open first.

## Which file is the second design option
The second design is here:

- `Codes/code 2/personal-site-option2.html` – the page
- `Codes/code 2/personal-style-option2.css` – the styles

It shows the same content with a different look.
