# Design Decisions

This file records the main design choices. Add a new note here whenever the
website changes.

## Colours
- **Red** (`#e10600` / `#ff2b2b`) – the main accent colour, used for highlights,
  buttons and section labels. It gives a racing / motorsport feel.
- **Black / very dark grey** (`#080808`, `#101010`) – used for backgrounds to make
  the photos and red accents stand out.
- **White / light grey** – used for text and for some bright sections so the page
  is easy to read.

## Layout style
- A modern, card-based layout.
- Sections alternate between dark and light backgrounds to keep it interesting.
- Grids are used for the driver cards, track cards and the gallery.
- The page is responsive: on small screens the columns stack into one column.

## Image choices and why
- **Hero (Design 1):** `tyre-perspective.jpg` – a strong, eye-catching shot for the
  first thing visitors see.
- **My Story section (Design 1):** `child-kart.jpg` – a childhood karting photo that
  matches the story about starting karting young.
- **Gallery:** a mix of action and portrait photos to show different racing moments.
- **Backgrounds:** `driving.jpg` and `walking-away.jpg` are used as faded background
  images behind dark overlays so text stays readable.

## Sponsor-friendly design decisions
- A clear **Sponsors** section explains what a sponsor would get.
- Professional-looking photos are shown large so brands can imagine their logos
  alongside them.
- Social media links (Instagram, TikTok) are easy to find for contact.
- Clean, simple design so the focus stays on the driver and the photos.
